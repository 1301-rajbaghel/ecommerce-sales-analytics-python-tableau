# E-Commerce Customer & Sales Analytics Using Python and Power BI

## Project Overview

This project analyzes a global e-commerce sales dataset containing more than 51,000 transactions to uncover customer behavior, product performance, sales trends, profitability drivers, and market performance. Python was used for data cleaning and exploratory data analysis (EDA), while Power BI was used to create interactive dashboards and business intelligence reports.

---

## Objectives

- Analyze customer purchasing behavior.
- Identify top-performing products and categories.
- Evaluate sales and profit performance across markets and regions.
- Understand the relationship between sales, profit, discounts, and shipping costs.
- Build an interactive Power BI dashboard for business insights.

## Key Insights

- Technology generated the highest overall sales and profit.
- APAC emerged as the highest revenue-generating market.
- England recorded the highest sales among all states/regions.
- Phones were the top-selling sub-category.
- Copiers generated the highest profit among all sub-categories.
- Standard Class was the most commonly used shipping mode.
- Higher discounts showed a negative impact on profitability.
- Sales increased consistently from 2011 to 2014.
