# E-Commerce Sales Analytics Using Python & Tableau

## Project Overview

This project analyzes a global e-commerce sales dataset containing over 51,000 transactions to uncover sales trends, customer behavior, product performance, and market-wise business insights.

Python was used for data cleaning and exploratory data analysis (EDA), while Tableau was used to build an interactive business dashboard.

---

## Tools & Technologies

* Python
* Pandas
* Jupyter Notebook
* Tableau Public
* CSV Dataset

---

## Key Analysis Performed

* Sales by Category
* Profit by Category
* Market-wise Sales Analysis
* Customer Analysis
* Product Performance Analysis
* Year-wise Sales Trend
* Shipping Mode Analysis
* Correlation Analysis

---

## Key Insights

* Technology generated the highest sales and profit.
* APAC was the top-performing market.
* Phones emerged as the best-selling sub-category.
* Copiers generated the highest profit.
* Sales showed consistent growth from 2011 to 2014.
* Higher discounts negatively impacted profitability.
